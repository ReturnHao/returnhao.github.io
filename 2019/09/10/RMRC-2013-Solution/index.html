B. Seven Kingdoms

Description

给你一张联通图，求一种分配方案，将原图分为三个互斥的联通分量，并且恰有两个联通分量分别包含城市序号为111，222的点，保证城市111和城市222不会直接相连，也不存在一个公共城市与它们两个相连。如果不存在，则输出impossible。

Solution

由于城市111，222之间没有公共城市，所以我们可以将图中除了特殊城市111，222的点分成以下三类：

  • 仅和城市111相连的城市集aaa
  • 仅和城市222相连的城市集bbb
  • 不和城市111相连，也不和城市222相连的城市集ccc

设包含城市111的联通分量为AAA，包含城市222的联通分量为BBB，其余点组成的联通分量为CCC。

由于图的特殊性，对上述三类的点进行进一步划分可知：

  • aaa中的点要么属于AAA，要么属于CCC
  • bbb中的点要么属于BBB，要么属于CCC
  • ccc中的点仅可能属于CCC

对于不属于集合CCC的点，根据它们和城市111和222的关系一定可以确定出它们属于AAA还是属于BBB（或者无解）。

于是考虑2-Sat，定义XiX_iX​i​​（True）为i∈(A∨B)i \in (A\lor B)i∈(A∨B)，Xi′X^{'}_iX​i​​′​​​​（False）为i∈Ci\in Ci∈C。

考虑所有没有边相连的点对{Vi,Vj}\{V_i, V_j\}{V​i​​,V​j​​}，根据其所属集合的不同共计四种关系：

  • Vi∈a(b)∧Vj∈a(b)V_i\in a(b) \land V_j\in a(b)V​i​​∈a(b)∧V​j​​∈a(b)：此时ViV_iV​i​​和VjV_jV​j​​不能同时属于A or BA \ or \ BA or B，亦不能同时属于CCC，即，Vi⊕Vj=1V_i\oplus V_j = 1V​i​​⊕V​j​​=1。
  • Vi∈a(b)∧Vj∈b(a)V_i\in a(b) \land V_j\in b(a)V​i​​∈a(b)∧V​j​​∈b(a)：此时ViV_iV​i​​和VjV_jV​j​​不能同时属于CCC，即Vi⊙Vj=1V_i\odot V_j=1V​i​​⊙V​j​​=1。
  • Vi∈a,b(c)∧Vj∈c(a,b)V_i \in a,b(c) \land V_j \in c(a,b)V​i​​∈a,b(c)∧V​j​​∈c(a,b)：此时ViV_iV​i​​必须为真（假），VjV_jV​j​​必须为假（真）。
  • Vi∈c∧Vj∈cV_i\in c \land V_j\in cV​i​​∈c∧V​j​​∈c：此时显然无解。

满足所有关系的2-Sat可以视作一组可行解，同时再根据预处理时点的所属集合，就不难确定整个方案了。

C. Institute of Advanced Category Manipulation

Description

给你nnn个三元组，问对于每一个三元组(xi,yi,zi)(x_i, y_i, z_i)(x​i​​,y​i​​,z​i​​)是否存在一组特定的常数(a,b,c)(a, b, c)(a,b,c)，满足a2+b2+c2=1a^2+b^2+c^2=1a​2​​+b​2​​+c​2​​=1，使得U_i=ax_i+by_i+cz_i=\underset{1\le j\le n}{\max}{U_j}

Solution

考虑空间坐标系，对于每一个特定的iii，我们有：

\forall j\not = i, a*(x_i-x_j)+b*(y_i-y_j)+c*(z_i-z_j)\geq0

从而我们必须找到一个平面(a,b,c,0)(a, b, c, 0)(a,b,c,0)使得点集P={(xi−xj,yi−yj,zi−zj)}P=\{(x_i-x_j, y_i-y_j, z_i-z_j)\}P={(x​i​​−x​j​​,y​i​​−y​j​​,z​i​​−z​j​​)}中的所有的点都位于这个平面之上。

假定我们已经找到了这样一个平面，我们可以通过平移和旋转，使得这个平面恰通过PPP中的两个点。

注意到这个平面必须通过(0,0,0)(0, 0, 0)(0,0,0)，于是，我们可以枚举PPP中的两个点{Pi,Pj}\{P_i, P_j\}{P​i​​,P​j​​}，并判断所有点是否位于这个平面之上。

另：所有点位于同一个平面的情况需要特殊判定。

D. Digital Content Protection

Description

给你一颗满二叉树，其中部分叶节点被污染了。规定一个点能选择，当且仅当它及它的子树上的所有点均未被污染，求一种选取方案，使得这些点的子树的叶节点集合恰等于原二叉树的叶节点集合并且选择的点尽可能的少。

Solution

要满足所需的点尽可能的少，就需要更多的选择内部节点。容易想到自下向上维护，贪心选取。然而nnn有646464，所以完全存储不太现实，考虑自顶向下，由于是满二叉树，所以对于某个点iii，很容易知道其叶节点的编号范围[l,r]=[i∗2depth,(i+1)∗2depth−1][l, r]=[i*2^{depth}, (i+1)*2^{depth}-1][l,r]=[i∗2​depth​​,(i+1)∗2​depth​​−1]。在这个范围O(m)O(m)O(m)扫描一遍是否有被污染的节点即可。

⚠️注意使用unsigned long longunsigned\ long\ longunsigned long long！
